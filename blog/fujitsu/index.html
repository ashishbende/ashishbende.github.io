Fujitsu Network Communications

At Fujitsu, I’m contributing to Virtuora-NC - an OpenDaylight based SDN controller. Over the years, I worked on different projects. Starting with the NorthBound, followed by MLPCE (where I spent most of my time) and, most recently, the Topology project.

Virtuora NC

  • Topology:

A module that discovers physical network inventory and builds topology representation. Developing a physical topology visualization feature across different abstraction layers to pinpoint exact network alarm events.

  • MLPCE:

A path computation engine responsible to create a service route across network resources. Implemented various constraint-based path calculation features and modeled a new set of network devices into system DBs to extend controller capabilities

  • NorthBound:

A user-facing GUI component, facilitating system interactions. Designed user request forms and API end points handling the request. Worked on translating vendor-neutral open-standard (OpenROADM) service requests to an internal representation.

Project details: Virtuora Network Controller

Tech Stack: Java, Python, JanusGraph/CassandraDB, OpenDaylight
