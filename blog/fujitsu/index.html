Fujitsu Network Communications

At Fujitsu, I’m contributing to an OpenDaylight based SDN (Software-defined networking) controller project - Virtuora-NC

Virtuora NC

Topology: A module that discovers physical network inventory and builds topology representation.

Developing a physical topology visualization feature across different abstraction layers to pinpoint exact network alarm events..

MLPCE: A path computation engine responsible to create a service route across network resources.

Implemented various constraint based path calculation features. Extended controller capability by modeling a set of new network entities.

NorthBound: A user-facing GUI component, facilitating system interactions.

Developed user request forms and API end points handling the request. Worked on translating vendor-neutral open-standard (OpenROADM) service requests to an internal representation.

Project details: Virtuora Network Controller

Tech Stack: Java, Python, JanusGraph/CassandraDB, OpenDaylight
