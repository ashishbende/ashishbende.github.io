Rackspace Summer Internship

I was an intern at Rackspace during the summer'16. I worked with the ObjectRocket team, a relatively small but dedicated team. I was tasked to develop an end-to-end data connector project for managed database service (MongoDB/Elasticsearch). A bit intermediating at the beginning, but I thoroughly enjoyed working on this feature.

In a short span, I developed the connector which would consume real-time Twitter feeds and store them in either MongoDB or Elasticsearch instance. Additionally, I coded a pre-defined Kibana dashboard template and POC product tour to enhance user experience.

Project details:Twitter-to-Elasticsearch connector

Tech Stack: Python, GO, Docker, Apache Mesos/Marathon, RabbitMQ
